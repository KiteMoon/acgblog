# 什么是云计算

云计算是一种商业模式，该商业模式提供信息技术类商品

例子：百度网盘

对于用户而言，不需要感知储存资源位置

不需要考虑技术

不需要考虑运维

对于用户而言，完全无感知

云计算的特点

  • 按需自助服务
  • 广泛网络接入
  • 弹性伸缩
  • 资源池化
  • 可计量服务

# 如何学习云计算

主要从两个角度

  • 如何实现云计算这种模式
    • 基础网络
    • 储存技术
    • 计算类技术
    • 操作系统
    • 数据库
    • 迁移
    • 备份
    • 容灾
    • 开发类技术
  • 怎么用？

# 云计算的三个时代

时代定义
1.0侧重于计算方向
2.0侧重于标准：制定出了一系列标准，这些标准在之后让行业更加统一（全世界 80% 的技术都是类似的）（OpenStack 就是这里出来）
3.0侧重于云原生，万物上云

# 云原生的定义

可以理解为万物上云的时代

# 云计算的四种部署模式

  • 公有云
    • 由大型企业或者运营商搭建的云数据中心对外提供服务，通过网络，公众均可使用，服务选择多，价格便宜
  • 私有云
    • 企业自建数据中心，为公司内业务提供服务，造价昂贵、服务数量不多，隐私性墙
  • 混合云
    • 某些场景中，业务需要同时使用公有云提供服务，也需要私有云提供服务，两种模式共同使用。
  • 行业云
    • 针对某些行业定制的云计算平台

# 云计算的服务模式

IAAS: 基础设施即为服务 (ECS)(云原生适用于这一层)

PAAS: 平台即为服务 (中间件一类的)（CDN)(开发者适用于这一层)

SAAS: 软件即为服务 (用成熟的软件给另一个软件使用)（比如数据分析服务给数据软件使用)(OSS / 分析服务)
