# FTP 基本操作

在说交换机相关的知识之前，我们最好还是先来学下 FTP 的操作

在所有的操作中，你的 put get 都是基于当前目录的，和 linux 差不多，具体看下面四个就行

put 上传单个文件

get 下载单个文件

mput 上传多个文件

mget 下载多个文件

# 路由器配置 FTP

无法理解，FLASH 才 32M 的空间，都需要拿来 FTP 了么。。。

首先启动路由器，进管理 CLI，然后进 aaa

system-view
aaa

然后传统艺能，创建一个用户，设置一个密码

[r1-aaa]local-user alice password cipher 123456

接着我们分配服务，分配的服务要选择 FTP

[r1-aaa]local-user alice service-type ftp

然后不一样的就来了。FTP 需要单独配置一个文件夹作为 FTP 的目录。因此。还要补充配置下

[r1-aaa]local-user alice ftp-directory flash:/

配置好了。我们就可以在另一台主机中连接的

# 这里是R2
<r2>ftp 192.168.2.1
Trying 192.168.2.1 ...
Press CTRL+K to abort
Connected to 192.168.2.1.
220 FTP service ready.
User(192.168.2.1:(none)):alice
331 Password required for alice.
Enter password:
230 User logged in.

到这里就完成 FTP 的配置了，需要注意的是，FTP 的传输走的是明文，如果有人在主机的前面加了个抓包的，你的密码就没了。

# 路由器配置相关

我们会发现，咱们的路由器关机后重启配置都没，这是因为我们所有操作都建立在内存中，如果不写入 FLASH，下次重启还是会丢失配置的，因此我们只需要执行一个命令

<r2>save

执行完毕后，会跳出一些提示，这些是关于是否覆盖配置文件、保存的文件名和后缀相关的

The current configuration will be written to the device.
Are you sure to continue?[Y/N]Y
Info: Please input the file name ( *.cfg, *.zip ) [vrpcfg.zip]:
Jul  5 2022 22:07:03-08:00 r2 %%01CFM/4/SAVE(l)[0]:The user chose Y when decidin
g whether to save the configuration to the device.
Now saving the current configuration to the slot 17.
Save the configuration successfully.

当然，如果我们要查看目前我们正在内存中生效的配置文件，就可以用下面这个命令查看

[r2]display current-configuration

与之对应，假如说我们 FLASH 中有一个配置文件，我们要读取他，就可以用下面这个命令了

[r2]display saved-configuration

如果我们需要对比他们，那么就可以使用下面这个命令

这个命令可以对比内存中的配置文件和 FLASH 中的配置文件的区别，方便检查

[r2]compare configuration

最后，假如说我们需要清除我们 FLASH 中的配置文件，就可以使用下面这个命令一键清空。

[r2]reset saved-configuration
