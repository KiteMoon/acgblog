# HCIE Day 4

# 存储设备

存储设备一般会采用存算分离

  • 速度快：网速一般是 10GE
  • 共享数据：多个服务器可以共享一个储存
  • 安全：服务器崩了其他也没事

# 负载均衡

用于将从一个入口 (IP) 流入的流量均衡的流向各个服务器

可以让多台服务器共同承载业务，实现均衡负载

软件可以使用：Nginx

硬件叫：F5

# LINUX 基础介绍

Linux 他是一个系统内核，他不是一个完整的系统

发行版是在 Linux 内核的基础上进一步安装各类软件

NB 的 linux：RedHat

华为的欧若拉就是基于红帽

# 割接

割接指的是大面积升级

极度危险，极度费心神

# 虚拟网络类型

# 桥接模式

为客户机操作系统提供直接访问外部以太网络的权限。客户机在外部网络上必须要有自己的 IP 地址

能上网，能被别人访问

简单理解，虽然他在虚拟机，但是他会生成一个模拟的网口，直接连接到外部网络，分的 IP 和物理机是一个网络号

# NAT

为客户机操作系统提供使用主机 IP 地址访问主机拨号连接或者外部以太网

能上网，不太好外部访问

类似于这样，用户请求 1.1.1.1 但是先要通过物理机 IP 192.168.2.5 ，这个时候，我们需要使用 192.168.2.5:6666 这个 IP + 端口，来映射到内部的 1.1.1.1 ，当然也可以映射到 IP + 端口，如 192.168.2.5:777->1.1.1.1:80

# 仅主机模式

只能在 Vmware 主机内部访问，需要在同一个 Vmware 网络中才能实现互访 物理机可以访问到。

不能上网，只能外部互访

同个网络下允许互访 (网络号一致)，本身宿主机也可以访问到，但是外部网络无法访问

# SCSI 小型计算机接口协议

一般用于 CPU - 内存 - 磁盘中间通信使用

内存：非持久存放数据

硬盘：持久存放数据

# 准虚拟化 SCSI

速度最慢

# LSI Logic

速度比较快，兼容性强

# LSI Logic SAS

速度和上面差不多，兼容性较差
