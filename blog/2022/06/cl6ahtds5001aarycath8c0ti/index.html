# 命令概述

首次提醒（之后不会有提醒了）: 每次这样的 Part 都会有这样的一个章节，该章节代表本 Part 内所有的命令集合，您可以通过查看这里来快速了解命令相关知识

进入系统界面: system-view

进入对应网络接口或者 console 口

(这里是进 G 口以太网 0/0/0 代表接口)

interface GigabitEthernet 0/0/0

修改 ip:

命令中的 24 代表子网掩码 (255.255.255.0)

ip address 192.168.1.2 24

查看 IP 地址:`

brief 带表查看所有接口，如果不添加 ip 则代表查看网口信息

进入用户 conosle 接口
进入用户 console 空是为了修改 console 密码

关闭消息提示:
友情提示，教学环境可以随便关，但是在生产环境就别随便关了。不然你没地方哭

显示当前接口中输入的命令:
显示你在当前位置输入过的历史命令

设置密码:
这里在设置密码模式，有三种模式，AAA（带账号和密码）password (纯密码模式)，none（无密码）

设置密码内容:
这个命令可以设置各种模式的密码，但是这里只展示纯密码模式（即为只输入密码，不需要输入账户）
password 代表输入密码为纯密码模式
simple 代表简单密码模式
123456 是密码内容

注意，无论明文密文，在网络上均为明文传输

# 给对应的接口设置 IP

首先，我们需要从初始化界面中进入系统界面，然后在系统界面中进入我们需要设置的接口

从初始界面进入系统界面

system-view

从系统界面进入网卡界面

interface GigabitEthernet 0/0/0

接着，我们就可以设置 IP 地址了

注意，后面的 24 代表子网掩码，你愿意的话可以用 255.255.255.0 代替

ip address 192.168.1.2 24

设置完毕后可以通过 display 命令检查你的 IP 是否配置正确

display ip interface brief

此时，如果你完成操作，应该就可以看到对应的 IP 从逻辑和物理上联通

# 关闭消息提示

前排提醒。强烈不推荐这样，除非是教学环境，否则一律不要关这个，不然后面炸了你都不知道咋回事

很简单，一行命令搞定

undo info-center enable

输入完成后，会回执你一条消息，代表关闭成功

Info: Information center is disabled.

# 修改 console 密码 [纯密码模式]

这个操作还是要学会的，防外还是要防内，不然别人改你机器你还在那里哭

从初始界面进入系统界面

system-view

从系统界面进入用户 console

user-interface console 0

进入后，我们先设置鉴权模式，这里我们选择纯密码模式，CLI 命令如下

authentication-mode password

设置完毕后，我们接着设置密码

这里需要注意下命令行细节

password 代表使用纯密码模式

cipher 代表使用加密模式输入密码，这个模式下，你使用 dis th 命令的时候，看到的密码就是乱码，与之对应的有 simple （不加密模式）, 但是请务必注意，无论加密与否，在网络上都是明文传输的

123456 代表密码

set authentication password cipher 123456

这样就设置密码成功了

# 查看历史命令

和 history 差不多，只不过命令变了，同时注意，他是显示你在当前接口下输入的命令，如果你跨区域了，就看不到了（比如从系统页面切换到接口页面，那么系统页面的内容就没了），命令如下

dis th
